# Assignment prompts (local)

This folder holds **example** rubric text for the repo. For real grading, create per-assignment files on your machine:

```
~/documents/grading/<assignment_name>/prompts/
├── context.txt      # Rubric + output JSON schema (you write this)
└── solutions.pdf    # Answer key (not committed to git)
```

See [HOW_TO_GRADE_NEW_ASSIGNMENT.md](../HOW_TO_GRADE_NEW_ASSIGNMENT.md) for the full format.
